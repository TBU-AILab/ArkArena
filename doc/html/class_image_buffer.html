Detailed Description

template<class T, int size = 2>
class ImageBuffer< T, size >

Template class for storing set of continusous data. The data are stored in array - the class is working like a circular buffer. The data in buffer can be be overwrite when the buffer is full or the push function can be blocking.

Template Parameters
TData type of stored values
sizeSize of circular buffer
